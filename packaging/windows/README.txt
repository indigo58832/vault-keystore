Vault для Windows — один файл
===============================

Запускайте только Vault.exe (двойной клик).

Внутри уже есть:
  - интерфейс склада ключей
  - сервер проверки ключей (localhost:17777)

KeyCheckerServer.exe и VaultLauncher.exe больше не нужны.

База ключей:
  C:\Users\<имя>\.local\share\keystore\keys.db

С Linux скопируйте keys.db в эту папку (создайте каталог keystore если нет).

Quick Check: Ctrl+Shift+S (или скопируйте ключ Ctrl+C и нажмите hotkey).

Если проверка не работает — подождите до 30 сек после первого запуска
(распаковка onefile). Внизу окна индикатор сервера должен стать зелёным.
